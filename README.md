# 🍽️ 송직한 리뷰
**송직한 리뷰**는 숙명여자대학교 객체지향프로그래밍 과목 프로젝트로, 학교 주변 식당의 할인 정보와 리뷰를 한데 모아 제공하는 플랫폼입니다. 인증 절차를 거친 학생들만이 리뷰를 작성할 수 있어, 믿을 수 있는 평가를 통해 사용자들이 학교 근처의 최고의 식당을 쉽게 찾을 수 있도록 하였습니다.

## 🧠 주요 기능
 
1. **회원가입 및 로그인**: 사용자(학생 인증 포함)와 식당 관리자(식당 주인) 두 가지 역할로 구분하여 가입 및 로그인
2. **음식 카테고리 탐색**: 한식, 일식, 중식, 아시안, 양식, 샐러드&요거트, 카페&디저트, 술집 등 카테고리별 식당 검색
3. **식당 리스트 조회**: 카테고리별 식당 목록을 별점, 리뷰 수, 주소, 영업시간 등과 함께 제공
4. **리뷰 보기**: 식당에 달린 리뷰를 최신순, 별점 높은 순, 별점 낮은 순으로 정렬하여 확인
5. **리뷰 작성**: 학생 인증을 마친 사용자만 리뷰 작성 가능
6. **리뷰 필터링**: 메뉴별, 특징별(위생 등) 필터를 통해 원하는 리뷰만 선별 조회
7. **관리자 마이페이지**: 식당 주인이 본인 가게 정보 등록·수정 및 리뷰 통계 확인
8. **가게 정보 등록**: 관리자가 이미지, 메뉴, 영업시간 등 가게 정보를 직접 등록

## ▶️ 실행 방법
 
1. **저장소 클론**:
```bash
   git clone https://github.com/SM-OOP-project/SM-OOP-project.git
```
 
2. **프로젝트 열기**:
   - Java 개발 환경(JDK 설치)이 필요합니다.
   - IDE(IntelliJ IDEA, Eclipse 등)에서 프로젝트를 열거나 터미널에서 컴파일합니다.
3. **실행**:
```bash
   javac LoginScreen.java
   java LoginScreen
```
 
4. **기능 사용**:
   - 사용자 또는 관리자(식당 주인)로 회원가입 후 로그인합니다.
   - 사용자는 음식 카테고리 탐색 → 식당 리스트 → 리뷰 보기/작성 순으로 이용합니다.
   - 관리자는 마이페이지에서 본인 가게 정보를 등록하고 리뷰 통계를 확인합니다.

## 💡 추가 정보
 
- **학생 인증 시스템**: 회원가입 시 학생 인증 절차를 거친 사용자만 리뷰를 작성할 수 있어 신뢰도 높은 리뷰를 보장합니다.
- **이중 사용자 구조**: 일반 사용자(학생)와 관리자(식당 주인)를 분리하여 각 역할에 맞는 화면과 기능을 제공합니다.
- **데이터 정렬 기능**: 리뷰를 최신순, 별점 높은 순, 별점 낮은 순으로 정렬하여 원하는 정보를 빠르게 탐색할 수 있습니다.
- **카테고리 & 특징 필터**: 메뉴 종류별, 위생 등 특징별 필터링으로 세밀한 리뷰 검색이 가능합니다.
